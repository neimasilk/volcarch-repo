# E054: Pan-Austronesian Cognacy Gradient — Continental-Scale Test

## Hypothesis
Peripheral Austronesian languages (far from the Java/Malay center) retain higher
rates of PMP cognates than central languages — confirming P9's peripheral conservatism
thesis at continental scale.

## Method
Calculated PMP cognacy rate for 1,309 Austronesian languages in ABVD (≥50 concepts each).
Tested correlation with distance from Java (cultural overwriting center) and Taiwan
(dispersal homeland). Used Spearman rank correlation.

## Results — INFORMATIVE (SPLIT)

### Global Scale: REVERSED
- Cognacy vs Distance-from-Java: **rho=-0.088, p=0.002** (NEGATIVE)
- Cognacy vs Distance-from-Taiwan: **rho=-0.117, p=0.00002** (NEGATIVE)
- Languages CLOSER to Java retain MORE PMP cognates at global scale
- Explanation: phylogenetic gradient dominates — Java is near PMP homeland

### Regional Means (ordered by cognacy):
| Region | N | Mean Cognacy | Distance from Java |
|--------|---|-------------|-------------------|
| Java | 25 | 41.6% | 431 km |
| Philippines | 24 | 39.8% | 2,051 km |
| Eastern Indonesia | 222 | 39.5% | 2,505 km |
| Borneo | 36 | 39.0% | 1,337 km |
| Bali/Lesser Sunda | 3 | 38.9% | 370 km |
| Sulawesi | 134 | 37.0% | 1,228 km |
| Oceania (Remote) | 485 | 34.7% | 6,219 km |
| Taiwan (Formosan) | 74 | 30.9% | 3,636 km |
| Madagascar | 42 | 30.3% | 6,743 km |
| Melanesia | 233 | 30.1% | 4,639 km |

### Local Scale: CONFIRMS E043
- **Balinese: 41.3%** > **Javanese: 33.8%** (+7.5%, matches E043's +7.3%)
- **Malagasy varieties: 38.2-41.2%** (mean 39.5%, close to Balinese)
- **Old Javanese: 56.7%** → modern Javanese 33.8% = **22.9% erosion** since ~900 CE
- **Javanese (Yogyakarta): 28.4%** — even lower than generic Javanese (court center effect)

### Key Insight: Two Gradients
1. **Phylogenetic gradient** (global): languages phylogenetically closer to PMP naturally
   retain more cognates — this is a tree-topology effect, not cultural
2. **Cultural overwriting gradient** (local): within Western Malayo-Polynesian, court
   centers (Java, especially Yogyakarta) lost more than peripheries (Bali, Madagascar)

P9's peripheral conservatism is a **local phenomenon** operating WITHIN the
global phylogenetic gradient. Both gradients are real; they operate at different scales.

## Validation
- Balinese 41.3% vs Javanese 33.8%: **matches E043** (40.3% vs 33.0%)
- Old Javanese 56.7%: **matches E043** (55.3%)
- Malagasy ~39.5%: **matches E043** (40.8% for Merina)
- Consistency across independent analysis confirms the pattern is robust

## Status: INFORMATIVE (SPLIT — global reversed, local confirmed)
