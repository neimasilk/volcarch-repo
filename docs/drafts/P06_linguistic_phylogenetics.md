## Hypothesis

Ancient Nusantaran texts (prasasti, kakawin, kidung, Lontara) can be clustered phylogenetically using a 6-dimensional feature vector (Sanskrit loanword density, PAn lexical retention, maritime semantic richness, PAn morphological retention, verb-initiality index, indigenous ritual vocab ratio) to reconstruct pre-Indic linguistic strata.
