# E048: Multi-Domain Temporal Convergence Analysis

## Hypothesis
If pre-Hindu Nusantaran culture was overwritten by a *wave* (not a permanent transformation),
then ALL evidence channels should show correlated temporal dynamics.

## Method
Merged all DHARMA datasets (E023, E030, E033, E035, E040) into one unified temporal matrix.
166 dated inscriptions, C6-C14.

## Results — SUCCESS

### Key Finding 1: Cross-domain correlation survives length control
- Pre-Indic vocabulary ↔ Organic material mentions: **rho=+0.546, p<0.0001**
- After controlling for inscription length: **partial rho=+0.162, p=0.038**
- Inscriptions that preserve pre-Indic language also describe an organic material world

### Key Finding 2: Genre Taphonomy quantified (L5)
- Short-format inscriptions: organic mentioned 24.1% of time
- Long sima inscriptions: organic mentioned 90.4% of time
- Mann-Whitney p<0.0001 — **genre is a massive taphonomic filter**
- Pre-Indic ratio also higher in long sima (0.179 vs 0.059)

### Key Finding 3: Organic mentions increase over time
- Spearman rho=+0.714, p=0.047 (century-level)
- As inscriptions shift to Old Javanese sima format, organic world becomes visible

### Key Finding 4: C8 is the "dark century"
- C8 inscriptions: 0.5% pre-Indic ratio, 12.7% mention organic materials
- C9-C10: pre-Indic rises to 15%, organic to 67-83%
- C8 = peak Sanskrit format → minimum visibility of indigenous culture

## Interpretation
The "archaeological blank" of pre-Hindu Java is partly a **genre artifact**:
Sanskrit-format inscriptions (C8) simply didn't record organic materials or pre-Indic terms.
The indigenous civilization didn't appear — it was always there but textually invisible.
