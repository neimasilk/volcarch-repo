# E034 — Cerita Panji in Malagasy Literature Search

**Status:** INFORMATIVE NEGATIVE
**Date:** 2026-03-10
**Idea ID:** I-004

## Hypothesis

If Cerita Panji (the Javanese prince narrative cycle) exists in Malagasy oral tradition, it would date Panji to pre-1200 CE — before Majapahit, since Austronesian migration to Madagascar was approximately 800-1200 CE. This would be a gold-standard dating anchor.

## Method

Systematic literature search via web (academic databases, folklore archives, comparative literature) for:
1. Direct Panji/Inao parallels in Malagasy oral tradition
2. Structural analogs (hero-rescues-betrothed motif) in Malagasy folklore
3. Known geographic distribution of Panji tales
4. Timeline of Panji crystallization vs. Malagasy migration

## Key Results

### 1. Panji tales are NOT found in Madagascar

No source — academic, folkloric, or popular — mentions Panji/Inao narratives in Malagasy tradition. The known distribution of Panji is:
- **Core:** Java (Kediri/Majapahit origin)
- **Inner spread:** Bali, Lombok, Sulawesi, Kalimantan, Sumatra
- **Outer spread:** Malay peninsula (as Hikayat), Thailand, Myanmar, Cambodia, Vietnam (as Inao)
- **Absent:** Madagascar, Polynesia, Philippines (no Panji attestation)

### 2. Panji POST-dates the Malagasy migration

| Event | Date |
|-------|------|
| Austronesian migration to Madagascar | ~800-1200 CE |
| Smaradahana kakawin (proto-Panji, Kediri) | ~1100-1150 CE |
| Earliest Panji temple reliefs (Candi Jawi) | 13th century |
| Major Panji spread (Majapahit period) | 14th-15th century |

The Panji cycle **crystallized after the main Malagasy migration**. Its absence in Madagascar is therefore EXPECTED and chronologically consistent.

### 3. Ibonia epic: the closest Malagasy structural analog

The Ibonia epic — Madagascar's most important oral narrative — shares structural elements:
- Hero (Ibonia) rescues betrothed (Rampela) from evil adversary
- Series of tests, duels, magical transformations
- Reunion and royal marriage that affirms lineage
- Contains "Indonesian-style riddles and poems" (Haring 2013)

However, scholars compare Ibonia to the **Ramayana**, not Panji:
- Rampela abducted by "Trouble-Stone Man" ≈ Sita abducted by Ravana
- Hero's quest across magical landscapes ≈ Rama's journey to Lanka
- The Ramayana reached Indonesia by 7th-8th century — BEFORE Malagasy migration

### 4. Dating bracket established

| Narrative Layer | In Malagasy? | Implied timing |
|----------------|-------------|----------------|
| Ramayana framework | YES (Ibonia) | Pre-migration (before ~1200 CE) |
| Panji cycle | NO | Post-migration (after ~1100 CE) |
| Islamic narratives | NO | Post-migration (after ~1300 CE) |

This creates a **narrative terminus ante quem**: Malagasy oral culture preserves pre-Panji Nusantaran elements but excludes post-Kediri innovations.

### 5. Bonus finding: Famadihana

Famadihana (Malagasy "turning of the bones" ritual) is described as "an adaptation of premodern double funeral customs from Southeast Asia and Oceania" (Wikipedia, citing Larson 2001). This directly supports P5/P9's argument that mortuary practices were transplanted from Nusantara.

## Interpretation

The absence of Panji in Madagascar is an **informative negative**:

1. **Chronological consistency:** Panji crystallized ~1100-1150 CE; main Malagasy migration was ~800-1200 CE. The two events barely overlap. Absence is expected.

2. **Narrative stratigraphy:** Malagasy oral tradition preserves OLDER Nusantaran layers (Ramayana-era, Austronesian riddle structures) but lacks NEWER layers (Panji, Islamic narratives). This is exactly what a ~1200 CE "time capsule" would look like.

3. **For P12 (mythology classifier):** Ibonia could be classified as Class B (Indic-influenced) or Class C (syncretic) — it has Ramayana structure but Austronesian poetic form. It would be a valuable calibration point for the classifier's syncretic category.

4. **For P9 (peripheral conservatism):** Madagascar as the ultimate "peripheral" — preserves pre-1200 CE Nusantaran elements frozen in time, including double burial customs (famadihana).

## Limitations

1. **Literature search only** — no access to Malagasy-language oral tradition databases or field collections. A Malagasy folklorist might identify deeper parallels.
2. **Ibonia comparison is structural, not textual** — no morphological analysis of the narrative.
3. **"Absence of evidence ≠ evidence of absence"** — Panji elements might exist in unstudied regional Malagasy traditions.
4. **Migration timing debated** — some evidence for multiple waves, with the latest possibly post-1200 CE.

## Conclusion

**INFORMATIVE NEGATIVE.** Panji is not found in Malagasy tradition. This is chronologically expected (Panji post-dates migration) and creates a valuable dating bracket: Malagasy oral culture = pre-1200 CE Nusantaran time capsule that includes Ramayana-era elements but excludes Panji and Islamic layers. The finding strengthens the "Malagasy as control group" methodology used in P5, P9, and P11.
