# E044: Malagasy Burial Botanical Survey

**Status:** SUCCESS
**Date:** 2026-03-11
**Paper:** P9 (Peripheral Conservatism)

## Hypothesis

If the Javanese botanical burial complex (aromatic trees at cemetery/burial sites) represents pre-Hindu Nusantaran practice, equivalent aromatic botanical elements should appear in Malagasy burial contexts (exported ~1200 CE, before Islamization).

## Method

Desk research: web search for ethnobotanical literature on Malagasy burial plants, famadihana botanical elements, sacred cemetery trees. Cross-reference with Javanese/Balinese equivalents.

## Key Findings

### Finding 1: Plumeria (Kamboja) is New World — NOT Pre-Hindu

**Critical correction to P9 draft §2.3:** Plumeria spp. (frangipani/kamboja) is NATIVE TO THE AMERICAS (Mexico, Central America, Caribbean). Introduced to Philippines by Spanish in the **1560s**, then spread to Indonesia/Malaysia.

**Implication:** Kamboja at Javanese Muslim cemeteries is a **post-1560 CE substitute**, NOT a pre-Hindu aromatic burial tree. The P9 "botanical substitution chain" must be revised:

```
Pre-Hindu Nusantara:
  → Exposed burial + NATIVE aromatic tree
  → Taru Menyan (Trunyan-specific) = surviving example
  → Canarium spp. (pan-Austronesian) = wider candidate

Post-Portuguese contact (~1560s):
  → Plumeria introduced from New World
  → Adopted at cemeteries because similar aromatic function
  → REPLACES whatever native tree was previously used

Result: Kamboja = 4-layer palimpsest (not 3):
  Layer 1: Pre-Hindu native aromatic (Canarium? Taru Menyan?)
  Layer 2: Indianization (function shifts from decomposition to purification)
  Layer 3: Islamization (inhumation reinforced)
  Layer 4: Portuguese contact (~1560) introduces Plumeria as substitute
```

### Finding 2: Canarium — The Pan-Austronesian Aromatic Candidate

**Canarium** (Burseraceae family) is the strongest candidate for the pre-Hindu aromatic burial tree:

| Species | Location | Use |
|---------|----------|-----|
| C. madagascariense (ramy/haramy) | Madagascar | Ceremonial incense, "freshen the house" |
| C. strictum (dammar/sambrani) | Indonesia | Religious ceremonies, incense |
| C. commune (Java almond) | Java | Native aromatic |
| C. indicum | Melanesia | "Oldest and most important tree crop," ceremonial facepaint, canoe caulk |

Canarium is noted as among the "oldest and most important tree crops in Melanesia" — an Austronesian-carried crop. Its ceremonial resin use spans the entire Austronesian range from Melanesia to Madagascar.

**The Canarium connection = Austronesian-era botanical link between Java and Madagascar in a ceremonial context.**

### Finding 3: Malagasy Sacred Burial Botany

- **Sacred forests (ala masina):** Trees around tombs MUST NOT be cut — 35 unique woody species in fady-protected tomb forests. Functional parallel to sacred groves around Javanese/Balinese temples.
- **Baobab as ancestral residence:** Mikea people: ancestors live in sacred trees (hazofaly). Dead babies placed in baobab branches, not buried. Structural parallel to Trunyan exposed burial.
- **Fady (taboo):** Botanical taboos protect specific species — etymologically and functionally parallel to Javanese larangan/pamali concepts.
- **Perfume/wine on remains:** Famadihana uses perfume sprayed on exhumed bones — aromatic component in mortuary practice persists.
- **Shrouds wrapped with "willow mats":** Organic plant wrapping material for remains.

### Finding 4: Trunyan Inscription (I-047 check)

Copper plate inscription at Trunyan temple: **833 Saka (~911 CE)** confirmed. Also bronze tablet from 911 CE at Pura Tegeh Koripan on Mt. Penulisan (Batur caldera). Temple predates inscription — architectural evidence older. No direct mention of burial practice in inscription sources found via web search; DHARMA corpus check still needed.

## Conclusion

**STATUS: SUCCESS — with critical revision to P9 framework.**

1. **Plumeria = New World plant → revise P9 §2.3.** The "botanical substitution chain" gains a fourth layer (Portuguese contact ~1560s), making it MORE interesting, not less. The question becomes: what NATIVE aromatic tree did kamboja replace?

2. **Canarium = strongest pan-Austronesian candidate.** Present in Indonesia AND Madagascar as ceremonial incense tree. Austronesian-carried crop. This is the bridge P9 needs.

3. **Structural homologies confirmed:** Sacred forests around tombs, botanical taboos, exposed burial in trees, aromatic treatment of remains — all present in both Nusantara AND Madagascar, supporting pre-Islamization Austronesian origin.

4. **No Plumeria in Malagasy burial** (expected — Plumeria hadn't reached Indian Ocean by 1200 CE). But Canarium IS present as ceremonial aromatic. This is MORE diagnostic than Plumeria would have been.
