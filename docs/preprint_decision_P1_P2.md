# Preprint Decision: EarthArXiv for P1 and P2

**Date:** 2026-03-12
**Status:** Decision memo for author review

---

## The Question

Should we post P1 (Taphonomic Framework) and P2 (Settlement Model) as preprints on EarthArXiv?

---

## PRO (Arguments FOR posting preprints)

### 1. Priority Establishment
- P1 submitted 2026-03-10, P2 submitted 2026-03-11
- Review process takes 2-6 months — during this time, anyone could publish similar work
- EarthArXiv preprint establishes a public timestamp + DOI within days
- Volcanic taphonomic bias is not a crowded field, but settlement predictive models ARE
- **Risk of being scooped:** Low for P1 (niche topic), moderate for P2 (ML + archaeology is trendy)

### 2. Cross-Citation with DOI
- **Critical dependency:** P5 (BKI) and P9 (JSEAS) both cite P1's sedimentation rates
- Currently cited as "Author, in review" — reviewer cannot verify the claim
- A preprint DOI (e.g., `doi:10.31223/X5XXXX`) allows reviewers of P5/P9 to CHECK the cited work
- This may prevent "provide published citation" revision requests

### 3. Visibility and Discoverability
- EarthArXiv is indexed by Google Scholar, Semantic Scholar, and some Scopus services
- Preprints generate citations even before journal acceptance
- For a new researcher (no h-index yet), early visibility accelerates recognition

### 4. Journal Policies Explicitly Allow It
- **Asian Perspectives:** No preprint policy stated, but UH Press journals generally allow preprints. Confirm with editor if concerned.
- **JCAA (Ubiquity Press):** Explicitly preprint-friendly. Their policy states: "Posting a preprint does not constitute prior publication." ✓

### 5. Community Feedback
- Preprints may attract informal peer review (Twitter/X, archaeology blogs)
- Could lead to collaborator interest — an archaeologist sees P2 and offers GPR partnership

---

## CON (Arguments AGAINST posting preprints)

### 1. Rare Reviewer Objection
- Some reviewers (especially older, traditional) view preprints as "prior publication" and may recommend rejection
- **Probability:** Very low for JCAA (explicitly preprint-friendly). Low-to-moderate for Asian Perspectives (no explicit policy).
- **Mitigation:** Check Asian Perspectives editorial policy first, or email editor to confirm

### 2. Locks the Text
- Once posted, the preprint text is public. If reviewers request major revisions, the preprint version becomes an outdated public record
- EarthArXiv allows version updates (v2, v3), but the original v1 remains accessible
- **Mitigation:** Post the SUBMITTED version. Label clearly as "submitted manuscript, not peer reviewed"

### 3. P2 Requires Co-Author Agreement
- P2 has Go Frendi Gunawan as co-author
- Preprint posting requires his explicit consent
- **Action needed:** Ask Go Frendi before posting P2. P1 (single-author) can be posted independently.

### 4. Administrative Overhead
- EarthArXiv requires OSF account, metadata entry, CC license selection
- Modest time investment (~1-2 hours per paper)
- **Not a real barrier** — just needs to be scheduled

### 5. Possible Confusion with arXiv P8
- P8 is already submitted to arXiv (cs.CL, submit/7351261, status "on hold")
- Having P1/P2 on EarthArXiv and P8 on arXiv creates a split preprint footprint
- **Not a real problem** — different platforms serve different communities (geoscience vs CS)

## Alternative: Wait Until First Decision

If uncomfortable with preprinting now, a middle path:
- Wait for the FIRST editorial decision on P1 or P2
- If "revise and resubmit" → post preprint THEN (text is reviewer-improved)
- If "accept" → journal will publish final version, preprint less necessary
- If "reject" → post preprint immediately while resubmitting elsewhere (establishes priority)

**Drawback:** Loses the cross-citation DOI benefit during the current review cycle.
