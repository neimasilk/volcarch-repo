## Technical Notes for Future Submissions

### eJournalPress (AP) Quirks
- Single file upload only — must upload one at a time
- Radio buttons don't work with `fill_form` — use `run_code` with element IDs
- File Order tab requires explicit checkbox confirmation
- "Validate PDF" must be clicked before approve checkbox becomes enabled
- Merged PDF is auto-generated but needs manual validation + approval
- JPG only for figures (PNG rejected)
