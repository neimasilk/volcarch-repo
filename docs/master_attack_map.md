## The Consilience Argument

William Whewell (1840) coined "consilience of inductions":
when multiple INDEPENDENT lines of evidence converge
on the same conclusion, that conclusion is established
with greater confidence than any single proof could provide.

Darwin used this for evolution.
Wegener used this for continental drift.
VOLCARCH uses this for pre-Hindu Nusantaran civilization.

```
CLAIM: Pre-Hindu Nusantaran civilization existed,
       was sophisticated, was volcanically shaped,
       and is recoverable despite taphonomic loss

STRENGTH: Not one proof — eleven independent channels
          Each channel = independent confirmation
          All eleven converging = consilience
```
