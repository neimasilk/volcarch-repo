# E053: Ancient DNA Taphonomic Gap in Island Southeast Asia

## Hypothesis
The near-total absence of ancient DNA (aDNA) from Java is itself evidence of volcanic
taphonomic bias. Volcanic soils (acidic, hot, high moisture) systematically destroy DNA,
creating a gap in the genetic record of the most populated island in the Austronesian world.

## Method
Compiled 21 aDNA site records from published literature across Island Southeast Asia,
mainland SE Asia, Madagascar, and Taiwan. Tested whether aDNA recovery correlates with
volcanic proximity using Fisher's exact test and Mann-Whitney U test.

## Results — SUCCESS

### The Java Gap:
- **7 Java sites**, 84 samples attempted → **0 success (0%)**
- Non-Java ISEA: 14 sites → 7 successes (50%)
- **Fisher exact p=0.047** — Java significantly worse

### Volcanic Proximity Predicts aDNA Failure:
- Successful aDNA sites: mean **490 km** from nearest active volcano
- Failed aDNA sites: mean **144 km** from nearest active volcano
- **Mann-Whitney p=0.002** (success sites farther from volcanoes)
- **Point-biserial r=0.487, p=0.025**

### Soil Type Pattern:
- "volcanic" soil types: **0% success** (0/7)
- "limestone" soil types: **40-100% success** (3/6)
- "burial" soil types: **100% success** (3/3)

### The Population Paradox:
- Java: 151.6M people, 500+ archaeological sites, 60% volcanic coverage → 0 aDNA
- Sulawesi: 19.4M people, 50 sites, 10% volcanic → 2 aDNA successes

### Key Finding (meta-taphonomic):
The absence of aDNA from Java is not "absence of evidence" but **evidence of absence** —
volcanic taphonomy systematically destroys the very material (DNA) that could prove
pre-Hindu Java had complex populations. This creates a circular trap:
"No aDNA → can't prove population → assume empty → civilization starts with India."

## Implications
- P1: aDNA gap = strongest meta-evidence for taphonomic framework
- P5: Organic decomposition accelerated by volcanic soil acidity
- P8: Language = the ONLY surviving substrate when DNA is destroyed
- P9: Peripheral regions (Madagascar, Taiwan) where DNA IS preserved = our best windows
