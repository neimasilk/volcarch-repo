## JSEAS-Specific Style Notes (for copyediting after acceptance)

These are house style requirements that are typically applied during copyediting, not required for initial submission:

- "Per cent" (two words) in running text, % in tables only
- Page number style: pp. 107-9 (not 107-109)
- Date format: 24 Aug. 1971, 18th century
- First mention of person: full name (e.g., "Anthony Reid" not "Reid")
- BCE and CE (not BC/AD) — already correct
- Subsequent footnote citations: surname + shortened title (not op. cit.)
- British English: -ise endings, -our endings — mostly done
